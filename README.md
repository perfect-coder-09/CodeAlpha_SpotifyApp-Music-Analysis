# Spotify Music Data Analysis

## About The Project
Spotify Music Data Analysis is a data analytics project that explores music data to find useful insights about songs, artists, and popularity trends.

## Project Purpose
The purpose of this project is to analyze Spotify data and present meaningful insights through an interactive Power BI dashboard.

## Technologies Used
- Power BI
- Excel
- Data Visualization
- Data Analysis

## Analysis Performed
- Analyzed song popularity
- Studied top performing artists
- Compared music categories
- Identified trends in Spotify data

## Dashboard Highlights
- Total Songs Overview
- Top Artists Ranking
- Popular Tracks Analysis
- Year-wise Music Trends
- Interactive Filters

## Files Included
- Spotify Power BI Dashboard
- Dataset
- Dashboard Screenshots
- Project Documentation

## Key Learnings
- Data cleaning and transformation
- Creating interactive dashboards
- Extracting insights from large datasets

## Conclusion
This project demonstrates how data analytics and visualization can be used to understand music trends and user preferences.
